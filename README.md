# Facial Emotion Recognition System using Deep Leaning Techniques

## Project Description
This project detects human emotions in real time using facial expressions.

## Objectives
- Detect faces using MTCNN
- Classify emotions using deep learning
- Real-time webcam emotion detection

##  Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- NumPy

## Algorithms Used
- Face Detection: MTCNN
- Emotion Classification: Mobilenetv2 Model

##  Dataset
CK+ Facial Expression Dataset

## How to Run
pip install -r requirements.txt
python realtime_emotion.py

## Results
Show emotion label on screen
Display mental state and recommendation
